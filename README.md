## Características.

- Desafio feito em linguagem Java.
- Requisições de APIs com Retrofit pois facilitam a manutenção e utilização.
- Dados armazenados em um Banco de Dados local (SQLite) e no Firebase(somente autenticação no Login).
- Navegação utilizando Fragments para maior fluidez.
- Layout minimalista com enfase em poucos clicks e UX, sem a necessidade de muitos textos explicando o que cada item faz.

## Carregando Cotações

Ao abrir o aplicativo as cotações das moedas "Brita (BRT)" e "Bitcoin (BTC)" são carregadas de suas respectivas APIs e colocadas em um banco de dados local SQLite.
